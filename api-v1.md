# Api Version 1

## Endpoints

### Login

```http
POST /v1/login
```

### Users

#### List all Users

```http
GET /v1/users
```

#### Get User

```http
GET /v1/users/{id}
```

#### Create User

```http
POST /v1/users
```

#### Update User

```http
PUT /v1/users/{id}
```

#### Delete User

```http
DELETE /v1/users/{id}
```
